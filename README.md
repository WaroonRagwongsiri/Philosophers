## *This project has been created as part of the 42 curriculum by waragwon*

> Project philosopher is all about **concurrent, process, threads, data race, race condition** by wanting as to recreate the ***DiningPhilosopher***

> DiningPhilosopher is problem where each table have ***n philosohper*** but only have ***n forks*** on the table to be able to eat the spagehtti each philosopher need ***2 forks***.  
If philosopher didn't eat in the predetermine time they will ***starve to dead*** philosopher will have this cycle ***think eat sleep***

### Mandatory
> The mandatory part is about the threads.  
We will represent each philosopher as threads.

### Bonus
> The bonus part is about the another concept of semaphore.

### Usage
#### Mandatory
> cd philo  
> make  
> ./philo number_of_philosophers time_to_die time_to_eat time_to_sleep [number_of_times_each_philosopher_must_eat]
#### Bonus
> cd philo_bonus  
> make  
> ./philo_bonus number_of_philosophers time_to_die time_to_eat time_to_sleep [number_of_times_each_philosopher_must_eat]
